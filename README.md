### Set UP
```
git clone -b reserve git@github.com:haohcraft/redux-boilerplate.git reserve
cd reserve
npm install
```

### Dev
```
npm run start
open `localhost:4000`
```


### Build
```
npm run build
```

### Demo

```
npm run build
cd build/
python -m SimpleHTTPServer 8000
open `localhost:8000`
```
### Note
This no icon for `cross` in your front folder, so I pick a warning icon instead
